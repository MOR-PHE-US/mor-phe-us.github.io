常见环境变量和基本配置

Morpheus2022/5/5大约 2 分钟WindowsnvmpythonadbJava

提示

重装 Windows 系统后需要安装一系列软件，配置环境变量时很麻烦，故而整理到一起，方便参考

配置指南中如 %NVM_HOME% 则是新建到系统变量后再添加到 Path 下

[\path\to] 只是一个占位符（placeholder），表示 “你电脑上这个文件夹实际所在的位置”。
它不是一个真实路径，需要你根据自己的系统，把它替换为 你的软件实际安装目录。

nvm

nvm 下载

nvm 及 node 配置

  • %NVM_HOME% 【nvm 环境变量】

    [\path\to]\nvm

  • %NVM_SYMLINK% 【node 环境变量】

    [\path\to]\nvm\nodejs

  • %NODE_PATH% 【npm 环境变量】

    [\path\to]\nvm\nodejs\node_global

  • settings.txt 【nvm 配置文件】

    root: [\path\to]\nvm
path: [\path\to]\nvm\nodejs
node_mirror: https://npmmirror.com/mirrors/node/
npm_mirror: https://npmmirror.com/mirrors/npm/

  • npm 【设置 node 全局和缓存路径命令】

    npm config set prefix "[\path\to]\node_global"
npm config set cache "[\path\to]\node_cache"

  • 【管理员：允许第三方软件运行 node 脚本】

    Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

  • 指定 .pnpm-store 存储路径【需要先安装 pnpm】

    pnpm config set store-dir [/path/to/].pnpm-store

其他安装

  • MacOS

    # Homebrew
brew install git

# MacPorts
sudo port install git

# Xcode Command Line Tools
xcode-select --install

# Installing git-gui
brew install git-gui

  • Linux

    # Debian/Ubuntu
apt-get install git
# 对于 Ubuntu，该 PPA 提供最新稳定的上游 Git 版本
add-apt-repository ppa:git-core/ppa
apt update
apt install git

# Fedora
yum install git # (up to Fedora 21)
dnf install git # (Fedora 22 and later)

# Gentoo
emerge --ask --verbose dev-vcs/git

# Arch Linux
pacman -S git

# openSUSE
zypper install git

# Mageia
urpmi git

# Nix/NixOS
nix-env -i git

# FreeBSD
pkg install git

# Solaris 9/10/11 (OpenCSW)
pkgutil -i git

# Solaris 11 Express, OpenIndiana
pkg install developer/versioning/git

# OpenBSD
pkg_add git

# Alpine
apk add git

# Slitaz
tazpkg get-install git

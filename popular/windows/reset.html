重装系统

Morpheus2023/7/22大约 2 分钟

重装前的准备（重要）（重要）（重要）

  • 完整备份个人文件（外接盘 / 云盘）。
  • 记录重要软件的序列号 / 登录凭证（如 Adobe、Office 等）。
  • 确认 Microsoft 账户是否与数字许可证关联（Settings → Activation）。
  • 取消 BitLocker 或暂时停用加密；如果使用硬盘加密，先解密。
  • 准备好驱动（网卡 / 显卡），尤其是没有网络驱动可能导致安装后无法联网的情况，可备用驱动 U 盘。
  • 如果有 RAID 或 NVMe 特殊驱动，提前下载并放入安装 U 盘中。
  • 确保有稳定电源（笔记本接电源），避免安装中断。

个人电脑常见重装方法

“重置此电脑”（Reset / 保留 / 删除）

设置 → 系统（或 更新与安全）→ 恢复 → 重置此电脑。
选项：

  • 保留我的文件（保留个人文件，删除应用和设置）
  • 删除所有内容

使用启动 U 盘 — 制作 Windows 安装 U 盘后从 U 盘全新安装。

  1. 在另一台电脑下载 Windows 镜像 ISO 或使用微软的 Media Creation Tool 制作可启动 U 盘（8GB+）。

  2. 在目标电脑进入 BIOS/UEFI（常见按键：F2、DEL、F12、Esc），设置从 USB 启动或使用启动菜单。

  3. 从 U 盘启动 → 在安装界面选择 “自定义安装” → 删除旧系统分区 / 格式化 → 安装。

厂商恢复分区 / 出厂恢复

使用厂商预置的恢复镜像（如 Dell/HP/Lenovo 提供的一键恢复）。

在 PE 系统中使用 Dism++ 释放到磁盘

  1. 选择 恢复功能 → 系统还原
  2. 选择 Windows 镜像文件
  3. 选择 要重装的磁盘
  4. 选择 目标映像 （家庭版 / 专业版）
  5. 勾选 添加引导 和 格式化
  6. 点击 确定
  7. 看到 映像已经还原成功 → 点击 确定 → 重启电脑
